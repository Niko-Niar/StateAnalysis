В папке StateAnalysis находиться основной код программы:
StateAnalysis - main
MosqConnect - подписка на mosquitto, прием и парсинг сообщений, а также отправка данных на анализ
Conversion - обрезка и интерполяция
BSAnalysis - сам анализ

Для работы проекта необходимо подключить библиотеку ormodule (папка libormodule2), libuv (папка libuv) и mosquitto (папка Subscribe)
1) В строке меню выбираем: Проект -> Свойства (Project -> Properties)
2)C/C++ -> Общие -> Дополнительные каталоги включаемых файлов (C/C++ -> General -> Additional Include Directories)
Добавляем туда путь до 3 папок libuv\include, libormodule2 и Subscribe
3)Далее Компоновщик -> Общие -> Дополнительные Каталоги Библиотек (Linker -> General -> Additional Library Directories)
Туда прописываем 2 библиотеки libuv и Subscribe
4)Теперь Компоновщик -> Ввод -> Дополнительные зависимости (Linker -> Input -> Additional Dependencies)
Туда записываем mosquitto.lib и libuv.lib
После этого должно заработать (если нет то мб чет не так сделали)

Для работы брокера mosquitto необходимо:
1) Скачать брокер mosquitto с официального сайта: https://mosquitto.org/
2) Открыть командную строку или терминал
3) Перейти в директорию с установленным mosquitto
4) Запустить mosquitto при помощи ввода команды: mosquitto
5) Запустить проект. В открывшемся окне должна появиться надпись "Подключение прошло успешно"
6) Открыть новое окно командной строки или терминала
7) Перейти в директорию с mosquitto и ввести команду: mosquitto_pub.exe -p 1883 -h localhost -t test -m 1000
8) В результате будет получено сообщение с числом 1000

Для тестирования использовались скрипты и датасеты находящиеся в папке Scripts
Скрипты отличаються друг от друга только файлом, который они посылают в конце записи
Во второй строке скрипта указан путь до папки с установленным mosquitto (при необходимости изменить)
В последней строке указан путь до файла датасета (У нас использовался путь C:\Logs\)
Для работы скриптов на Windows использовалась программа cygwin (https://cygwin.com/)
1)Скачиваем и устанавливаем программу
2)Открываем ее
3)С помощью команды cd переходим в папку со скриптами
4)Используем синтаксис ./script1.sh для запуска скрипта
Запускаем скрипт после того, как наша программа подключилась к mosquitto

При работе с Windows возникает ошибка из-за формата конца строк
Исправляется преобразованием формата в Unix
Для этого использовалась программа Notepad++ (https://notepad-plus-plus.org/)
Открываем наш скрипт, в строке меню выбираем: Правка -> Формат Конца Строки -> Преобразовать в Unix
